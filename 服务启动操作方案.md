# KnowQ 服务启动操作方案

## 一、环境要求

| 环境 | 版本要求 |
|------|----------|
| Node.js | >= 18.x |
| Python | >= 3.10 |
| npm | 随 Node.js 安装 |

---

## 二、后端服务启动

### 1. 进入后端目录
```powershell
cd d:\SVNServer\TraeCN\KnowQ\backend
```

### 2. 创建虚拟环境（首次启动）
```powershell
python -m venv venv
```

### 3. 激活虚拟环境
```powershell
.\venv\Scripts\Activate
```

### 4. 安装依赖（首次启动）
```powershell
pip install -r requirements.txt
```

### 5. 启动后端服务
```powershell
python -m uvicorn main:app --host 0.0.0.0 --port 6428 --reload
```

启动成功后访问：
- API 文档：http://localhost:6428/docs
- 默认管理员账号：`admin` / `admin123`

---

## 三、前端服务启动

### 1. 打开新终端，进入前端目录
```powershell
cd d:\SVNServer\TraeCN\KnowQ\frontend
```

### 2. 安装依赖（首次启动）
```powershell
npm install
```

### 3. 启动开发服务器
```powershell
npm run dev
```

启动成功后访问：http://localhost:2428

---

## 四、一键启动脚本

在项目根目录创建 `start.bat` 文件：

```batch
@echo off
start "KnowQ Backend" cmd /k "cd /d d:\SVNServer\TraeCN\KnowQ\backend && .\venv\Scripts\Activate && uvicorn main:app --host 0.0.0.0 --port 6428 --reload"
timeout /t 3
start "KnowQ Frontend" cmd /k "cd /d d:\SVNServer\TraeCN\KnowQ\frontend && npm run dev"
echo KnowQ 服务启动完成！
echo 前端地址: http://localhost:2428
echo 后端地址: http://localhost:6428/docs
pause
```

双击 `start.bat` 即可同时启动前后端服务。

## 六、常见问题

### 1. 端口被占用
```powershell
# 查看端口占用
netstat -ano | findstr "6428"
netstat -ano | findstr "2428"

# 结束占用进程（PID 为查询到的进程ID）
taskkill /PID <进程ID> /F
```

### 2. Python 虚拟环境激活失败
```powershell
# 如果遇到执行策略限制，先执行：
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
```

### 3. 依赖安装失败
```powershell
# 后端：使用国内镜像源
pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple

# 前端：清除缓存后重新安装
npm cache clean --force
npm install
```
